## 已建立的分类体系

### 清酒

**法定分类（特定名称酒）**，用于记录酒款基础信息：

| 类别 | 精米步合 | 特点 |
|------|---------|------|
| 純米大吟醸 | ≤50% | 最高等级，全米酿造 |
| 大吟醸 | ≤50% | 最高等级，含酿造酒精 |
| 純米吟醸 | ≤60% | 全米酿造，吟醸香 |
| 吟醸 | ≤60% | 含酿造酒精 |
| 特別純米 | ≤60%或特殊制法 | 全米，特别标示 |
| 純米 | 无规定 | 全米，无添加酒精 |
| 特別本醸造 | ≤60%或特殊制法 | 含少量酿造酒精 |
| 本醸造 | ≤70% | 含少量酿造酒精 |

**SSI四类型**，用于快速定位风格：

| 类型 | 香气 | 味道 | 典型代表 |
|------|------|------|---------|
| 薫酒 | 高 | 淡 | 大吟醸系、十四代 |
| 爽酒 | 低 | 淡 | 普通酒、本醸造 |
| 醇酒 | 低 | 浓 | 生酛、山廃、燗酒向け |
| 熟酒 | 高 | 浓 | 古酒、熟成系 |

**品评维度**（简化自SAKE DIPLOMA）：
- 外观：清澄度、色调
- 香气：浓度（第一印象）、特征（从固定词汇选取）
- 味道：重量感、甘み、酸み、苦み、バランス、余韵长度
- 结论：SSI类型判断、特定名称推断（如条件允许）

## 记录规范

**文件命名**：`YYYY-MM-DD_蔵元名_酒款名.md`

**评分**：★★★ / ★★ / ★ / —（不使用百分制）

**语言**：中文为主，日文专业术语保留原文（如：精米步合、吟醸香）

**香气词汇**：优先使用SAKE DIPLOMA标准词汇，可附加自己的描述

## 给Claude的工作指引

1. **新增酒款记录**时，使用 `docs/tasting-templates/sake-tasting.md` 模板
2. **更新探访场所**时，同步更新本文件的"已探访场所"表格
3. **建议品评词汇**时，优先用SAKE DIPLOMA标准用语，其次用SSI四类型框架
4. **不要**用评分系统以外的方式评价（比如"这款酒很棒"），用具体描述代替
5. 如果用户喝了新品类，先讨论是否建立新的分类体系，再开始记录
