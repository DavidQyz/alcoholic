# 06 · 後処理

> 压榨后的酒液还要经过一系列处理，才能成为你手里那瓶酒。这一步决定了酒的"状态"——贴标上出现最多术语的环节。

---

## 後処理の全流程

```
上槽（压榨）
   ↓
[濾過]  ← 可省略 → 無濾過
   ↓
[火入れ①]  ← 可省略 → 生酒 / 生貯蔵酒
   ↓
貯藏・熟成（数月～数年）
   ↓
[加水]  ← 可省略 → 原酒
   ↓
[火入れ②]  ← 可省略
   ↓
瓶装出荷
```

每个"可省略"的环节，就会在标签上留下一个词。

---

## 各环节详解

### 濾過（ろか）

压榨后的酒液仍有轻微浑浊和颜色，用**活性炭**过滤去除。

| 状态 | 标签词 | 效果 |
|------|-------|------|
| 正常过滤 | 无特别标注 | 清澈透明，风味更干净 |
| **無濾過（むろか）** | 無濾過 | 保留更多颜色（淡黄）、风味物质、层次感 |
| **おりがらみ** | おりがらみ | 保留微量酒粕颗粒，略有浑浊，旨味更强 |
| **にごり酒** | にごり / 濁り | 粗滤，大量固体保留，米甜感强，外观乳白 |

---

### 火入れ（ひいれ）——巴氏杀菌

加热至约60～65℃，杀死残余酵母和杂菌，停止发酵，稳定酒质。通常进行两次（贮藏前和出荷前）。

| 状态 | 标签词 | 特点 |
|------|-------|------|
| 两次火入れ | 无特别标注 | 最稳定，常温保存，保质期长 |
| **生酒（なまざけ）** | 生 | 完全未杀菌，最鲜活，香气最直接，**必须冷藏** |
| **生貯蔵酒（なまちょぞうしゅ）** | 生貯蔵 | 贮藏前不杀菌，出荷前杀菌一次，介于两者之间 |
| **生詰め（なまづめ）** | 生詰 | 贮藏前杀菌，出荷前不杀菌，保留出荷时的鲜活感 |

> 生酒是清酒中保质期最短、对运输要求最高的类型，买到后尽快饮用。

---

### 貯藏・熟成

大多数清酒在压榨后需要**贮藏约半年**才出荷，让风味充分融合稳定（"熟成"过程）。

| 类型 | 熟成时间 | 风格变化 |
|------|---------|---------|
| 新酒（しんしゅ） | 当年压榨即出荷 | 鲜活，有时带生酒感 |
| 普通熟成 | 半年左右 | 圆润，风味融合 |
| **古酒（こしゅ）** | 3年以上 | 琥珀色，焦糖・蜂蜜・木质香，风格接近雪莉 |

你喝过的**倉本酒造 菩提阮古酒2014**是超长期熟成的极端案例——酒瓶上标明年份，风味已完全进入熟酒（じゅくしゅ）领域。

---

### 加水（かすい）

压榨出来的原酒酒精度约17～20%，通常加水调整到14～16%再出荷。

| 状态 | 标签词 | 特点 |
|------|-------|------|
| 正常加水 | 无特别标注 | 口感更柔和，饮用更顺畅 |
| **原酒（げんしゅ）** | 原酒 | 未加水，酒精感强，风味浓缩，厚重 |

---

## 标签解读：叠加术语

一款酒可以同时有多个后处理词叠加，例如：

**「無濾過生原酒」**
- 無濾過 → 未炭素过滤
- 生 → 未杀菌
- 原酒 → 未加水

= 最"原始"状态，风味最浓郁直接，保质期短，需冷藏。

**「生詰め純米吟醸」**
- 生詰め → 贮藏前杀菌，出荷前不杀菌
- 純米吟醸 → 全米，精米60%以下

= 保留了一定鲜活感，但比纯生酒更易保存。

---

## 总结：看标签判断酒的状态

| 你想要 | 关注词 |
|--------|-------|
| 最鲜活、果香直接 | 生酒、生、無濾過 |
| 浓郁、厚重 | 原酒、無濾過 |
| 稳定、易保存 | 无特别标注（火入れ済） |
| 长期熟成的复杂感 | 古酒、熟成、年份标注 |
| 米甜、浑浊感 | にごり、おりがらみ |
