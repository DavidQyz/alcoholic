# 清酒知识基础 · Sake Primer

> 本文件是个人品饮参考，非考证用途。体系基于SAKE DIPLOMA / SSI唎酒師框架，有所简化。

---

## 一、清酒是什么

清酒（日本酒）是以**大米**为原料，经过**糖化**（米曲菌将淀粉转化为糖）和**发酵**（酵母将糖转化为酒精）的酿造酒，酒精度通常在14~16%之间。

和葡萄酒不同，清酒的发酵是糖化和发酵**同时进行**的（並行複醗酵），这是清酒酿造最独特的技术特征。

---

## 二、法定分类：特定名称酒

日本法律规定了八种"特定名称酒"，其余的统称"普通酒"（一般酒）。

### 核心参数

**精米步合**：大米被磨去的比例。50%意味着磨去外层50%，只剩内芯50%。
- 数字越小 → 磨得越多 → 杂味越少 → 香气越清澈
- 数字越大 → 保留外层 → 米的旨味更足 → 风格更厚重

**醸造アルコール**：酿造酒精（食用酒精）。少量添加可以提引香气，降低成本。
- 含有 → 非"純米"系
- 不含 → "純米"系

### 八种特定名称

```
                精米步合 ≤50%          精米步合 ≤60%         精米步合无规定
纯米系  │  純米大吟醸（最高）  │    純米吟醸 / 特別純米   │     純米酒
────────┼───────────────────────────────────────────────────────────────
添加系  │     大吟醸         │    吟醸 / 特別本醸造     │    本醸造酒
```

> **记忆捷径**：有"純米"→不加酒精；有"吟醸"→香气华丽；数字越小→越精致

## 三、风格分类：SSI四类型

由日本酒サービス研究会（SSI）提出，用"香气"和"味道"两个轴定位风格。

```
              高香气
               │
        薫酒   │   熟酒
  （大吟醸系） │ （古酒・熟成系）
               │
淡味 ──────────┼────────── 浓味
               │
        爽酒   │   醇酒
  （本醸造・普通酒）│ （生酛・山廃系）
               │
              低香气
```

| 类型 | 读法 | 代表款型 | 适饮温度 | 搭配料理 |
|------|------|---------|---------|---------|
| 薫酒 | くんしゅ | 大吟醸、純米大吟醸 | 冷酒（5-15℃） | 刺身、白身鱼 |
| 爽酒 | そうしゅ | 本醸造、普通酒、にごり酒 | 冷～常温 | 炸物、轻食 |
| 醇酒 | じゅんしゅ | 純米酒、生酛、山廃 | 常温～燗酒 | 烤物、味噌料理 |
| 熟酒 | じゅくしゅ | 古酒（5年以上熟成） | 常温～温燗 | 中华料理、蓝纹芝士 |

> 十四代、新政、獺祭等现代名酒大多属于**薫酒**，这是当代清酒市场最受欢迎的风格。

## 四、酿造工艺关键词

了解这些词，能帮助理解一款酒为什么是这个味道。

### 酒母（酛）类型——影响风味基础

| 类型 | 做法 | 风格倾向 |
|------|------|---------|
| 速醸酛 | 添加乳酸，快速安全 | 清洁、稳定，现代主流 |
| 生酛（きもと） | 自然乳酸菌，传统费时 | 厚重、复杂、有野性感 |
| 山廃（やまはい） | 生酛的简化版 | 介于两者，旨味饱满 |

### 处理方式——影响新鲜度与风格

| 词汇 | 含义 |
|------|------|
| 生酒（なまざけ） | 完全未加热杀菌，最鲜，需冷藏 |
| 生一本（きいっぽん） | 单一酒厂纯米酒 |
| 原酒（げんしゅ） | 未加水稀释，酒精度偏高（17-20%） |
| 無濾過（むろか） | 未经炭素过滤，保留更多风味 |
| にごり酒 | 未完全过滤，酒体浑浊，米甜 |
| 古酒（こしゅ） | 3年以上熟成，琥珀色，复杂 |

## 五、品饮温度与称呼

清酒拥有全世界最细致的饮用温度体系：

| 温度 | 称呼 | 特点 |
|------|------|------|
| 5℃ | 雪冷え | 香气收敛，清爽 |
| 10℃ | 花冷え | 吟醸系最佳区间 |
| 15℃ | 涼冷え | 香气开始展开 |
| 20℃ | 冷や / 常温 | — |
| 30℃ | 日向燗 | 米の旨味浮现 |
| 35℃ | 人肌燗 | 柔和，生酛系适合 |
| 40℃ | ぬる燗 | 醇酒最佳区间 |
| 45℃ | 上燗 | 辛口更辛，甘口更甜 |
| 50℃ | 熱燗 | 香气冲，需强劲款 |

> **实用原则**：薫酒→冷酒；爽酒→冷～常温；醇酒→燗酒；熟酒→常温或温燗

| 产地 | 特点 | 代表銘柄 |
|------|------|---------|
| 灘・伏見（兵庫・京都） | 历史最悠久，硬水/软水各异 | 白鶴、月桂冠、澤の鶴 |
| 山形 | 果实香丰富，现代风格 | 十四代、出羽桜 |

## 七、常用品评词汇（SAKE DIPLOMA 标准）

### 香气特征词

**果実系**：グレープフルーツ、黄色いリンゴ、洋ナシ、白桃、バナナ、メロン、マスカット、ライチ

**花系**：スイカズラ、スミレ、アカシア、菩提樹

**植物系**：青竹、新緑、セルフィーユ

**米・穀物系**：炊いた米、つきたての餅、上新粉、白玉団子、杏仁豆腐

**乳製品系**：生クリーム、サワークリーム、ヨーグルト、発酵バター、クリームチーズ

**木・スパイス系**：ヒノキ、香木、月桂樹の葉、丁子、シナモン

**熟成系**：紅茶、カラメル、コーヒー、醤油、蜂蜜、ビターチョコレート

**土・鉱物系**：粘土、腐葉土、石灰、ヨード香

### 味道形容

**甘み**：弱い / 上品な / まろやかな / ふくよかな / 力強い

**酸み**：爽やかな / 優しい / 丸みがある / シャープな / 力強い

**苦み**：控えめな / コク（深み）を与える / 旨味をともなった

**バランス**：スムースな / 溌剌とした / ドライな / まろやかな / 豊潤な / 力強い

**余韻**：短い / やや短い / やや長い / 長い
