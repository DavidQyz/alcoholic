# 威士忌品鉴记录

按 **产国 / 产区 / 酒厂** 三级组织：

```
whisky/records/
  苏格兰/
    Islay/
      Lagavulin/
        ★★★_2018_Lagavulin_16年.md
      产区概览.md
    Speyside/
      ...
  日本/
    （产区概念弱，中间一级可用产国或地区占位）
```

- 每款酒一个文件，命名 `评分_YYYY_酒厂_酒款名.md`（年份可用酒龄或装瓶年）
- 每个产区可建 `产区概览.md`（模板见 `whisky/templates/产区概览.md`）
- 品鉴模板：`whisky/templates/whisky-tasting.md`
