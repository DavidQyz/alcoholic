# 威士忌基础参考

> 这个品类**以记录为主**。我没有系统学过威士忌品饮，主要靠多喝 + 套用其他饮料的风味框架。
> 这份 primer 不求考证级，只提供组织记录所需的产区/类型对照，可逐步补充。

---

## 产区对照（用于 `records/产国/产区/酒厂/` 的中间一级）

| 产国 | 主要产区 | 大致风格印象 |
|------|---------|------------|
| 苏格兰 | Speyside | 果香花香、优雅，常见雪莉/波本桶 |
| 苏格兰 | Islay | 泥煤、烟熏、海风碘味 |
| 苏格兰 | Highland | 跨度大，从清爽到厚重 |
| 苏格兰 | Lowland | 轻盈、青草、谷物 |
| 苏格兰 | Campbeltown | 咸味、油脂感、少许烟熏 |
| 爱尔兰 | （通常不细分） | 三次蒸馏、顺滑 |
| 美国 | Kentucky / Tennessee | 波本为主，香草、焦糖、橡木 |
| 日本 | （多按酒厂） | 精致平衡；产区概念弱，产区一级可填酒厂所在地或「日本」 |
| 其他 | 加拿大 / 印度 / 台湾 等 | 逐案补充 |

> 产区概念弱的产国（如日本、爱尔兰），中间一级可直接用产国名或主要地区占位，不必强求。

## 类型速查

| 类型 | 说明 |
|------|------|
| 单一麦芽 Single Malt | 单一酒厂、100% 发芽大麦 |
| 调和麦芽 Blended Malt | 多家酒厂的麦芽威士忌调和 |
| 调和 Blended | 麦芽 + 谷物威士忌调和 |
| 单一谷物 Single Grain | 单一酒厂的谷物威士忌 |
| 波本 Bourbon | 美国，≥51% 玉米，全新炭化橡木桶 |
| 黑麦 Rye | ≥51% 黑麦 |
