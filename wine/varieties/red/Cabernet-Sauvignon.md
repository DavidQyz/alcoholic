# Cabernet Sauvignon · 赤霞珠

## 基本特征

| 项目 | 内容 |
|------|------|
| **皮厚** | 厚 |
| **酸度** | 高 |
| **单宁** | 高（需要暖气候才能成熟，否则单宁粗涩） |
| **酒体** | 中→饱满 |
| **颜色** | 极深 |
| **适宜气候** | 中等→温暖 |
| **橡木桶** | 常过桶（软化单宁）；新桶增加烟熏/香草/丁香/雪松 |
| **陈年特征** | 干果、泥土、森林地表 |

## 风味特征

- **标志性**：黑加仑、黑樱桃
- **草本**：青椒、薄荷（凉爽气候/未充分成熟时）
- **全熟**：煮熟黑果、果酱感
- **桶陈**：雪松、烟熏、香草
- **瓶陈**：干果、泥土、森林地表、烟草

## 为什么要混酿？

单宁高、需要暖气候，天气多变时难以每年都完全成熟 → 与 Merlot（软化单宁、增加红果）等混酿

## 重要产区

### 法国·波尔多左岸
- **Haut-Médoc**：Margaux、Pauillac、Saint-Estèphe
- **Graves / Pessac-Léognan**：偏暖，结构强，可陈年
- 天气多变，混酿（赤霞珠为主+梅洛+品丽珠等）

### 美国·加州
- **Napa Valley**：气候稳定，极熟，高单宁，饱满有力；Oakville、Rutherford、Calistoga 各有特色
- **Sonoma**：稍凉，更优雅

### 智利
- **Maipo Valley / Colchagua**：和梅洛混酿，薄荷+青椒+黑果+桶味

### 南非·Stellenbosch
质量高，从鲜果到饱满煮果都有，高酒精

### 澳大利亚
- **Margaret River**：熟黑果+桶味，质量高
- **Coonawarra**：红土（Terra Rossa），突出黑加仑+烟雪茄（桶味），可陈年

### 新西兰·Hawke's Bay
最热产区才种得了，混梅洛，红黑果+青椒辣椒
