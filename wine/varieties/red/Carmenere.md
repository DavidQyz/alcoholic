# Carmenère · 佳美娜

## 基本特征

| 项目 | 内容 |
|------|------|
| **单宁** | 高 |
| **颜色** | 深 |
| **适宜气候** | 温暖（需要很热才能完全成熟） |
| **橡木桶** | 通常过桶 |
| **陈年特征** | 皮革、泥土 |

## 风味特征（随成熟度差异极大）

| 成熟度 | 风味 |
|--------|------|
| **未充分成熟** | 辛辣草本：土豆叶、青椒——容易误认为 Cabernet |
| **充分成熟** | 黑果（黑莓）、饱满感 |
| **过桶** | 咖啡、巧克力 |

## 来历

- 原产法国波尔多，根瘤蚜病后几乎绝种
- 被带到智利后长期被误认为是 Merlot
- 1994 年才被正式确认为独立品种

## 重要产区

### 智利（唯一主要产区）
- 在智利中央谷广泛种植
- 需要温暖气候才能成熟，即便如此也很难完全成熟
