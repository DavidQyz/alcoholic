# Gamay · 佳美

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **单宁** | 低→中 |
| **酒体** | 轻→中 |
| **颜色** | 深（与薄皮相比颜色意外地深） |
| **适宜气候** | 中等 |
| **橡木桶** | 不过桶 |
| **陈年特征** | 一般不适合陈年 |

## 风味特征

- 红果（红樱桃、红李、覆盆子）
- **标志性**：糖果感、香蕉味（来自二氧化碳浸渍法工艺）

## 博若莱工艺：二氧化碳浸渍法（Carbonic Maceration）

- 目的：最大化颜色，最小化单宁
- 整串葡萄在密封罐中进行胞内发酵
- 结果：单宁极低，颜色深，果味鲜，出现糖果/香蕉特征风味
