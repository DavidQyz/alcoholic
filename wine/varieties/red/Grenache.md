# Grenache / Garnacha · 歌海娜

## 基本特征

| 项目 | 内容 |
|------|------|
| **皮厚** | 薄 |
| **酸度** | 低 |
| **单宁** | 低→中 |
| **颜色** | 浅（皮薄） |
| **酒精** | 高（高糖分品种） |
| **适宜气候** | 温暖（比赤霞珠更喜热） |
| **橡木桶** | 最好的过桶，一般不过桶 |
| **陈年特征** | 泥土、肉感、干果 |

## 风味特征

- 红果（红樱桃、草莓、红李）
- 白胡椒、甘草（有时）
- 陈年：干果、泥土、肉感

颜色浅、酸低、单宁低 → 经常与 Syrah（加颜色/酸/单宁）等混酿

## 重要产区

### 法国·南罗纳河谷
- **Côtes du Rhône**：简单便宜，与西拉等混酿
- **Villages**：更浓缩复杂
- **Châteauneuf-du-Pape**：饱满，高酒精，充分成熟；瓶陈有干果/泥土/肉感

### 西班牙（Garnacha）
- **Rioja DOC / Navarra DO**：鲜红果
- **Priorat DOCa**：老藤，极浓缩，混酿，瓶陈有干果/焦糖
