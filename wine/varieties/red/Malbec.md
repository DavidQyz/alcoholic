# Malbec · 马尔贝克

## 基本特征

| 项目 | 内容 |
|------|------|
| **单宁** | 高 |
| **酒体** | 饱满 |
| **橡木桶** | 通常过桶 |
| **陈年特征** | 干果、肉感 |

## 风味特征

- 黑果（黑莓、黑李）
- 桶陈：丁香、香草
- 瓶陈：干果、肉感

## 重要产区

### 阿根廷·Mendoza（高海拔）
- 几乎全世界最好的 Malbec 都在这里
- 高海拔（800-1,500m）保持酸度，昼夜温差大，浓缩度高
- 单宁高但果味充沛
