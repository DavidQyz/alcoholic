# Merlot · 梅洛

## 基本特征

| 项目 | 内容 |
|------|------|
| **皮厚** | 比赤霞珠薄 |
| **酸度** | 中 |
| **单宁** | 中 |
| **酒体** | 轻（刚熟）→饱满（充分成熟） |
| **适宜气候** | 中等→温暖 |
| **橡木桶** | 通常过桶（软化&增复杂度）或不过桶 |
| **陈年特征** | 干果、烟草 |

## 风味特征

- **较未熟**：新鲜红果（草莓、红李）+ 草本（青椒、辣椒）
- **充分成熟**：黑莓、黑李、饱满感
- **桶陈**：香草、椰子、烟
- **瓶陈**：干果、烟草

## 与赤霞珠的关系

经常混酿：梅洛增加红果感、降低单宁、提升柔顺度；两者是波尔多混酿的核心组合

## 重要产区

### 法国·波尔多右岸（主要以梅洛为主）
- **Saint-Émilion / Pomerol**：梅洛主导，饱满酒体，黑果集中，过桶；瓶陈有干果/烟草

### 美国·加州
- **Napa Valley / Sonoma**：果味浓，饱满，常有明显桶味（香草/椰子/烟）

### 智利·Central Valley
容易成熟，高产，柔和，中酒体

### 南非·Stellenbosch
复杂，陈年潜力好

### 澳大利亚
多用于和赤霞珠混酿，如 Margaret River

### 新西兰·Hawke's Bay
从轻到饱满都有，单一或混酿
