# Pinot Noir · 黑皮诺

## 基本特征

| 项目 | 内容 |
|------|------|
| **皮厚** | 薄 |
| **酸度** | 高 |
| **单宁** | 低→中 |
| **酒体** | 轻→中 |
| **适宜气候** | 凉爽→中等（要求最苛刻的品种之一） |
| **橡木桶** | 谨慎使用（小桶） |
| **陈年特征** | 蘑菇、森林地表（腐叶）、皮革、野味 |

## 风味特征

- **年轻**：红果（草莓、覆盆子、红樱桃）
- **陈年**：蘑菇、森林地表、皮革、烟草、泥土

皮薄→单宁低→颜色浅，容易过萃，酿造难度极高。

## 重要产区

### 法国·勃艮第（Bourgogne）
全球黑皮诺最高标杆，有极详细的分级体系（详见产区文件）

- Côte de Nuits：最顶级红酒，Gevrey-Chambertin、Vosne-Romanée
- Côte de Beaune：以白酒为主，红酒如 Pommard 也很好

### 美国·加州
- **Los Carneros**：凉，轻盈，红果
- **Sonoma Coast**：高酸，细腻
- **Santa Barbara**：质量高，复杂

### 美国·Oregon（俄勒冈）
凉爽，高酸，接近勃艮第风格

### 新西兰·Central Otago
极凉，高酸，集中，红果到黑果

### 德国·Baden / Pfalz（Spätburgunder）
酒体较轻，果味，风格近勃艮第
