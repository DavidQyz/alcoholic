# Pinotage · 皮诺塔吉

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **单宁** | 中 |
| **酒体** | 中 |
| **适宜气候** | 温暖 |
| **橡木桶** | 过桶后风格极重 |

## 来历

- 专门为南非温暖气候培育的杂交品种
- 亲本：**Pinot Noir**（黑皮诺）× **Cinsault**（神索）
- 1925 年在南非 Stellenbosch 大学培育

## 风味特征

- **不过桶**：红果（草莓、覆盆子、红樱桃）
- **过桶后**：咖啡、巧克力、烟味极重——这是 Pinotage 最有争议的特点

## 重要产区

### 南非·Western Cape
- 单酿版本：果味+过桶带来的重烟咖啡味
- **Cape Blends**：混入赤霞珠和梅洛，平衡更好
