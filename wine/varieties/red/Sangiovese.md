# Sangiovese · 桑娇维塞

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **单宁** | 中→高 |
| **酒体** | 中→饱满 |
| **适宜气候** | 温暖（意大利中部） |
| **橡木桶** | 传统大桶或现代小桶都有 |
| **陈年特征** | 皮革、烟草、泥土、干樱桃 |

## 风味特征

- 红果（红樱桃、红李）
- 明显酸度（标志性高酸）
- 单宁较重
- 陈年后：皮革、烟草、泥土、干玫瑰

## 重要产区（几乎全在意大利托斯卡纳）

| 产区 | 分级 | 风格 |
|------|------|------|
| **Chianti Classico DOCG** | Gran Selezione/Riserva/Annata | 桑娇维塞为主（≥80%），高酸高单宁，红樱桃 |
| **Brunello di Montalcino DOCG** | 顶级，100%桑娇维塞克隆（Brunello）| 饱满浓缩，极佳陈年潜力，最少5年才上市 |
| **Vino Nobile di Montepulciano DOCG** | 高端 | 类似 Chianti 但更饱满 |
| **Morellino di Scansano DOC** | 较平易 | 风格柔和，果味更突出 |

## 克隆差异

桑娇维塞有大量克隆品系，各地叫法不同：
- **Brunello**（蒙塔尔奇诺）
- **Prugnolo Gentile**（蒙特普尔恰诺）
- **Morellino**（马雷玛）

克隆不同，风格差异较大。
