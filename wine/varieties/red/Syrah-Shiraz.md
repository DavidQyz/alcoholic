# Syrah / Shiraz · 西拉

## 基本特征

| 项目 | 内容 |
|------|------|
| **皮厚** | 厚 |
| **酸度** | 中→高 |
| **单宁** | 中→高 |
| **酒体** | 中→饱满 |
| **适宜气候** | 中等→温暖 |
| **橡木桶** | 常过桶（皮革/肉/泥土感） |
| **陈年特征** | 干果、皮革、肉感、泥土 |

## 两种气候风格对比

| 气候 | 风味 |
|------|------|
| **凉→中等（Syrah）** | 新鲜黑果（黑樱桃/黑莓）+ 草本 + **黑胡椒**（标志性） |
| **温暖（Shiraz）** | 煮熟黑果 + 甘草 + 高酒精 |

北罗纳河谷有人说有种独特的"血/肉"气息，种在陡峭花岗岩坡面。

## 重要产区

### 法国·北罗纳河谷（Syrah 单酿）
- 种在陡坡（机器无法进入）
- **Côte-Rôtie**：最北最陡，最复杂，有黑胡椒感，极佳陈年潜力
- **Hermitage / Crozes-Hermitage**：地势平缓，产量较大，Hermitage 更集中

### 澳大利亚·南澳·Barossa Valley（Shiraz）
- 老藤低产，极浓缩
- 饱满，高单宁，煮熟黑果（黑莓/黑樱桃），黑胡椒，明显桶味

### 澳大利亚·新南威尔士·Hunter Valley
- 中酒体，中高单宁，新鲜黑果
- 瓶陈出泥土、肉感
