# Tempranillo · 丹魄

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 中 |
| **单宁** | 中 |
| **酒体** | 中→饱满 |
| **适宜气候** | 中等→温暖 |
| **橡木桶** | 传统用新桶（香草/烟/雪松） |
| **陈年特征** | 干果、皮革、蘑菇 |

## 风味特征

- 红果（草莓、红樱桃）
- 黑果（黑莓、黑李）
- 桶陈：香草、烟、雪松
- 瓶陈：皮革、蘑菇

## Rioja 陈年分级

| 等级 | 要求 |
|------|------|
| Crianza | 至少 2 年，其中 1 年桶陈 |
| Reserva | 至少 3 年，其中 1 年桶陈 |
| Gran Reserva | 至少 5 年，其中 2 年桶陈 |

## 重要产区（几乎全在西班牙）

### 西班牙·Rioja DOC
- 混酿为主（丹魄+歌海娜；或丹魄+其他加酸单宁）
- 简单款早饮；Gran Reserva 可长期陈年

### 西班牙·Ribera del Duero DO
- 饱满酒体，新鲜黑果

### 西班牙·Cataluña（加泰罗尼亚）
- 从简单果味到桶陈强劲都有
