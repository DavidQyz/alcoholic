# Zinfandel / Primitivo · 仙粉黛

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 中→高 |
| **单宁** | 中→高 |
| **酒体** | 中→饱满 |
| **酒精** | 高（喜暖气候，糖分高） |
| **适宜气候** | 温暖 |
| **橡木桶** | 红酒通常过桶 |
| **陈年特征** | 泥土、肉感 |

## 两种形态

| 形态 | 产地 | 风格 |
|------|------|------|
| **White Zinfandel**（桃红）| 美国加州 | 低酒精，中甜，简单，大量生产 |
| **红 Zinfandel** | 美国加州 | 高酒精，饱满，明显红黑果，过桶 |

## 风味特征

红色 Zinfandel：
- 红果→黑果（随成熟度变化）
- 高酒精带来的饱满热感
- 桶陈：香草、烟

## 重要产区

### 美国·加州（Zinfandel）
- 经典款（红）：高酒精，饱满，明显果味+桶味
- White Zinfandel（桃红）：大众市场，低酒精，微甜

### 意大利·Puglia（Primitivo）
- 柔和，果味，干型，高产；好的也很贵
