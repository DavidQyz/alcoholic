# Albariño · 阿尔巴利诺

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **酒体** | 中等 |
| **适宜气候** | 中等 |
| **橡木桶** | 不过桶 |
| **陈年特征** | 不适宜陈年 |

## 风味特征

- 柑橘：柠檬、葡萄柚
- 核果：桃子、杏
- 有时有轻微酵母感

## 重要产区

### 西班牙·Rías Baixas DO（西班牙西北角）
- 唯一核心产区
- 大西洋气候，多雨，高酸度
