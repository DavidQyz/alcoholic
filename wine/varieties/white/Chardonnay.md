# Chardonnay · 霞多丽

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 中→高 |
| **酒体** | 轻盈→饱满（高度可塑） |
| **适宜气候** | 凉爽→温暖 |
| **橡木桶** | 可过桶 / 可不过桶 |
| **陈年特征** | 榛子、蘑菇 |

## 风味特征（随气候变化）

| 气候 | 风味 |
|------|------|
| 凉爽 | 青苹果、柠檬、清爽酸度 |
| 中等 | 桃子、香瓜、核果 |
| 温暖 | 香蕉、菠萝、热带水果 |

## 酿造工艺对风味的影响

- **苹果酸乳酸发酵（MLF）**：降酸，增加黄油、奶油质感
- **酒泥接触（Lees Contact）**：增加酵母感（饼干、面包），提升酒体
- **橡木桶**：香草、椰子（美国桶）/ 雪松、烟熏（法国桶）；过桶后陈年出榛子、蘑菇

## 重要产区

### 法国·勃艮第（唯一法定白葡萄品种）

| 产区 | 气候 | 风格 |
|------|------|------|
| Chablis | 凉 | 轻→中酒体，青苹果/柠檬，无桶，高酸 |
| Côte de Beaune（Meursault/Puligny）| 中等 | 有酒体，核果，桶陈，陈年榛子蘑菇 |
| Mâconnais | 偏暖 | 果味（柠檬/桃/瓜），多不过桶；Pouilly-Fuissé 质量较高 |

### 法国·Pays d'Oc IGP
温暖高产，热带水果（菠萝/香蕉），简单便宜

### 美国·加州
- **Napa Valley**：桃子、菠萝，中酸，饱满酒体，常过桶
- **Los Carneros / Sonoma**：保留酸度，更复杂的水果
- **Santa Barbara County**：质量高，类似 Côte de Beaune

### 澳大利亚
- **Adelaide Hills / Yarra Valley**：凉，核果→热带，中高酸，少量过桶
- **Margaret River**：质量高，中→饱满酒体，桃子/菠萝

### 新西兰
- **Marlborough**：柠檬、桃、瓜，高酸
- **Hawke's Bay**：饱满酒体，高酸，明显桃子

### 智利
- **Casablanca**：柠檬、桃，有桶味
- **Central Valley**：便宜，高产

### 南非
- **Western Cape**：高产，常和白诗楠混
- **Walker Bay**：质量高，高酸，桃子/菠萝
