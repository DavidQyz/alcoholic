# Chenin Blanc · 白诗楠

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **酒体** | 轻→饱满（极高可塑性） |
| **适宜气候** | 凉爽→温暖 |
| **橡木桶** | 都有 |
| **陈年特征** | 干果、蜂蜜 |

## 风味特征（随成熟度/甜度变化）

- **干型/凉爽**：青苹果、柑橘、白花
- **半干/半甜**：核果（桃）、热带水果
- **贵腐/极甜**：热带水果、蜂蜜、干果

## 重要产区

### 法国·卢瓦尔河谷·Vouvray
- 干型：苹果、柑橘，不过桶
- 半干：用过熟或贵腐葡萄，核果或热带水果
- 全部不过桶

### 南非·Western Cape
- 气候较热，风格更果味（核果、热带）
- 产量大，价格亲民
