# Furmint · 富尔民特

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **酒体** | 饱满 |
| **适宜气候** | 中等 |
| **橡木桶** | 都有 |
| **陈年特征** | 干果、焦糖、坚果 |

## 风味特征

- 干果、橘子酱
- 陈年后：焦糖、坚果（榛子）

## 与贵腐酒的关系

Furmint 极易感染贵腐菌（Botrytis），是 **Tokaji Aszú** 的核心品种。

- **Puttonyos**：标签上的数字（3-6），代表每 136L 基酒中加入多少筐（puttony）贵腐葡萄——数字越大越甜

## 重要产区

### 匈牙利·Tokaj（托卡伊）
全球最著名的贵腐甜酒产地之一，传统上陈年后出售
