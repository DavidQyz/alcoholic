# Gewurztraminer · 琼瑶浆

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 低→中 |
| **酒体** | 饱满 |
| **适宜气候** | 凉爽→中等 |
| **橡木桶** | 一般不过桶；好的款过桶有蜂蜜和干果 |
| **陈年特征** | 蜂蜜、干果 |

## 风味特征

- **标志性**：荔枝——闻到荔枝十有八九就是它
- 核果（桃子、杏）
- 玫瑰花瓣
- 辛香料感

## 重要产区

### 法国·阿尔萨斯（唯一核心产区）
- 有 Grand Cru 分级
- 同时也种 Riesling 和 Pinot Gris
- 可做干型到贵腐甜型
