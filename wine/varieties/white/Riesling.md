# Riesling · 雷司令

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **单宁** | — |
| **酒体** | 轻→中 |
| **适宜气候** | 凉爽→中等 |
| **橡木桶** | 不过桶 |
| **陈年特征** | 蜂蜜、汽油/煤油（极具标志性） |

## 风味特征

风味变化极大，从干型到极甜都有：
- **年轻**：青苹果、白花、柠檬、核果（桃）、菠萝
- **陈年**：蜂蜜、汽油/煤油（Petrol）——是雷司令独有的三类香气标志

## 甜度类型（德国）

| 类型 | 风格 |
|------|------|
| Kabinett | 轻盈，微甜或干型 |
| Spätlese | 晚收，更熟，更甜 |
| Auslese | 精选晚收，甜 |
| Beerenauslese / Trockenbeerenauslese | 贵腐/过熟，极甜 |
| Eiswein | 冰酒，高糖高酸 |

## 重要产区

### 德国

| 产区 | 风格 |
|------|------|
| Mosel | 极凉，轻盈，中甜，低酒精，青苹果/柠檬 |
| Rheingau | 更有酒体，偏干 |
| Pfalz | 较暖，干型，中等酒体 |

### 法国·阿尔萨斯（Alsace）
- 干型为主，中等酒体，比德国更饱满
- 有 Grand Cru 分级

### 澳大利亚·Clare Valley / Eden Valley
- 干型，中高酸，柠檬/青柠，陈年出汽油/烤面包味
