# Sauvignon Blanc · 长相思

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 高 |
| **酒体** | 轻→中 |
| **适宜气候** | 凉爽→中等 |
| **橡木桶** | 通常不过桶 |
| **陈年特征** | 少，不适宜长期陈年 |

## 风味特征

- **凉爽气候**：草本（青草、黑加仑叶）、青苹果、柠檬、白醋栗
- **中等气候**：更多热带水果（百香果、芒果）
- **标志性**：青椒/草本味极突出（尤其新西兰 Marlborough）

有时与 Sémillon 混酿（增加酒体和陈年潜力）

## 重要产区

### 法国·卢瓦尔河谷

| 产区 | 风格 |
|------|------|
| Sancerre | 唯一合法白葡萄，凉，高酸，绿色水果和草本，精致 |
| Pouilly-Fumé | 类似 Sancerre |
| Touraine | 果味更多，价格更低 |

### 法国·波尔多
- AOC 基础款：简单，草本、青苹果
- Graves / Pessac-Léognan：更浓缩，常与 Sémillon 混酿

### 新西兰·Marlborough
草本和青椒味极为突出，轻酒体，高酸——最具辨识度的产区风格

### 澳大利亚
- **Margaret River**：浓缩草本，质量高
- **Adelaide Hills**：风格多变

### 智利
- **Casablanca Valley**：质量高
- **Central Valley**：高产，质量一般

### 南非
- **Constantia**：气候较凉，有青苹果味
- 整体偏热，多热带水果

### 美国·Napa Valley
天气偏热，有时过桶（增加烟熏味）
