# Sémillon · 赛美容

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 中→高 |
| **酒体** | 轻→饱满 |
| **适宜气候** | 中等→温暖 |
| **橡木桶** | 视产区而定 |
| **陈年特征** | 坚果、干果、蜂蜜 |
| **特点** | 极适合做贵腐甜酒 |

## 风味特征

- **年轻**：苹果、柠檬、青草
- **陈年**：坚果（榛子）、蜂蜜、干果、蜂蜡

## 重要产区

### 法国·波尔多·Sauternes（苏玳）
- **贵腐甜酒**：杏、桃、蜂蜜、橙皮酱，过桶
- 常与 Sauvignon Blanc 混酿（约 20-25%）

### 澳大利亚·Hunter Valley
- 极特殊：干型，轻酒体、低酒精；年轻时简单，但陈年 10-20 年后出蜂蜜、烤面包、坚果——全球少见的陈年潜力

### 澳大利亚·Barossa Valley
干型和甜型都有
