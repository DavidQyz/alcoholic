# Viognier · 维欧尼

## 基本特征

| 项目 | 内容 |
|------|------|
| **酸度** | 低→中 |
| **酒体** | 中→饱满 |
| **适宜气候** | 中等 |
| **橡木桶** | 很少过桶（偶尔轻微） |
| **酒精** | 通常偏高 |

## 风味特征

- **浓郁花香**：忍冬、紫罗兰、橙花
- **核果**：桃子、杏

## 重要产区

### 法国·北罗纳河谷·Condrieu
- 种在陡坡上，产量极低，价格高
- 略过桶（少量香草感）
- 全球最好的 Viognier 产地
