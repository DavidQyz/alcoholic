# 葡萄酒品评入门

## 框架来源

基于 **WSET Level 3 系统品评法（SAT）**。
二级用3档刻度；三级的核心升级是把大部分维度扩展为**5档刻度**（加入 Medium- / Medium+），并新增鼻腔状态、香气发展阶段、以及正式的品质结论框架（BLIC）。

---

## 二级 vs 三级的核心差异

| 维度 | 二级 | 三级 | 变化 |
|------|------|------|------|
| 外观：清澈度 | 无 | Clear / Hazy | 新增 |
| 外观：白酒颜色 | 柠檬/金/琥珀 | 加柠檬绿 | 扩展 |
| 香气：状态 | 无 | Clean / Unclean | 新增 |
| 香气：浓度 | 3档 | 5档（加 Medium± | 精细化 |
| 香气：发展阶段 | 无 | Youthful→Tired | 新增 |
| 口感：甜度 | 4档 | 6档（加 Medium-dry/sweet）| 扩展 |
| 口感：酸/单宁/酒精/酒体/浓度/余韵 | 3档 | 5档 | 精细化 |
| 结论：品质 | 5档（无Faulty）| 6档（加Faulty独立）| 扩展 |
| 结论：适饮表述 | 简单 | 4选项，需写完整短语 | 正式化 |

---

## BLIC 品质判断框架

| 字母 | 英文 | 含义 |
|------|------|------|
| B | Balance | 甜/酸/单宁/酒精/酒体相互协调，无单一元素突兀 |
| L | Length | 余韵中有多种愉悦风味持续，越长越好 |
| I | Intensity | 香气和风味浓郁清晰，有集中度 |
| C | Complexity | 一级/二级/三级香气同时存在，层次丰富 |

品质判断：满足0→Poor / 1→Acceptable / 2→Good / 3→Very Good / 4→Outstanding
有明显缺陷 → Faulty（独立于以上标准）

## 适饮表述（必须写完整句，否则不得分）

- **Drink now: not suitable for ageing** — 已到饮用高峰，不会因陈年改善
- **Can drink now, but has potential for ageing** — 现在可喝，结构支持继续发展
- **Too young** — 尚未到高峰，需要更多时间
- **Too old** — 已过高峰，走下坡

---

## 支持陈年的结构特征

高酸度 / 高单宁（红）/ 高残糖（甜酒）/ 高浓度 / 浓郁风味 / 平衡的结构
